  1. Проверим гипотезу о согласии наблюдаемх значений и модели посредством Hosmer-Lemeshow goodness-of-fit test.

H0: нет различий между предсказанными значениями вероятности и наблюдаемыми значениями в отдельных ранжированных подгруппах.

Для данного теста выполняется следующее:

  1. Вычисляется \(P(success)\) для каждого наблюдения, далее это значение ранжируется от наименьшего к наибольшему.

  2. Выборка разбивается на N-ное количество подгрупп: в качестве N берется 10 или N > k+1, где k – количество предикторов.

  3. По каждой подгруппе вычисляется статистика, она представляет собой разницу между наблюдаемыми и предсказанными значениями:

\(\sum_{j=1}^{k}\sum_{i=1}^{N}\frac{(Observed_{i}-Expected_{i})^2}{Expected_{i}}\)

hl <- hoslem.test(logit$y, fitted(logit), g=10)
hl.groups <- data.frame(cbind(hl$observed,hl$expected))
hl.groups <- mutate(hl.groups, total = y0 + y1)

\(H_{0}\): наблюдение относится к группе неголосовавших.

Ошибка первого рода:

\(error_{1} = P(reject|H_{0}) = \frac{FP}{FP+TN}\)

# er1 = 1 - 0.04061 
er1 <- FP/(FP+TN)
er1
## [1] 0.9593909

Следовательно, вероятность отнесения человека к группе голосовавших в то время, как он принадлежит к группе неголосовавших, то есть вероятность “ложноположительного срабатывания” равна 0.96.

Ошибка второго рода:

\(error_{2} = P(NOTreject|H1) = \frac{FN}{FN+TP}\)

# 1 - 0.98031
er2 <- FN/(FN+TP)
er2
## [1] 0.01968912

Следовательно, вероятность отнесения человека к группе неголосовавших в то время, как он принадлежит к группе голосовавших, то есть вероятность “ложноотрицательного срабатывания” равна 0.02.

Мощность:

\(Power = P(reject|H1) = 1 - \frac{FN}{FN+TP}\)

power <- 1 - er2
